# smail

### Project setup

1. **Download sdl2**

2. **Clone the repository to your local machine:**

   ```shell
   git clone https://github.com/YacineMK/smail
   ```
3. **download libs**

   ```shell
   sudo dnf install SDL2
   ```
   
   ```shell
   sudo dnf install SDL2_image-devel
   ```
   
   ```shell
   sudo dnf install SDL2_image-devel
   ```

4. **Run code**

   ```shell
   gcc startpage.c -o myprograme -lSDL2 -lSDL2_ttf -lSDL2_image
   ```

   ```shell
   ./mymyprograme
   ```
